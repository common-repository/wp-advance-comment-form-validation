== Description ==

This Plugin use for Wordpress Comments Validation to the Comment Form. Easy to Install and Need to Activate The Plugin.

= Features =
1. Easy to install.
2. No need to Settings in Admin Panel.

= Requirements =
This plugin requires PHP 5 or PHP 7 and WordPress 3.0 or greater. It works in standard WordPress environments. It is strongly recommended to use WordPress in the newest version.

== Installation ==

1. Go to Plugins > Add New. 
2. Searh plugin keyword "wp advance comment form validation".
3. install and activate the WordPress Plugin.
